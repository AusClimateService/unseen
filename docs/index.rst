unseen
======

A Python package for the analysis of large forecast ensembles.

Such analysis typically involves the UNSEEN
(UNprecedented Simulated Extremes using ENsembles;
`Thompson et al 2017 <https://doi.org/10.1038/s41467-017-00275-3>`__)
approach to assessing the likelihood of extreme events.
